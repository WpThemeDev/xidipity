Keyboard Shortcuts

  • ctrl-a - select all
  • ctrl-c - copy selected
  • ctrl-f - find
  • ctrl-v - paste from clipboard
  • [tab] - indent out
  • shift-[tab] - indent in

Important

Changes are not saved when the browser tab/window is closed.
